# Session 2 I2C examples

| Sketch | Target | Purpose |
| --- | --- | --- |
| [`arduino_slave/arduino_slave.ino`](arduino_slave/arduino_slave.ino) | Arduino Uno | Receives text at I2C address 9 and prints it to Serial |
| [`esp8266_master/esp8266_master.ino`](esp8266_master/esp8266_master.ino) | ESP8266 | Reads Serial input and sends it to I2C address 9 |

The recovered agenda describes removing selected expressions before the
exercise so learners can fill them in. The copies here preserve the complete
reference implementations. Confirm voltage compatibility, board pins and the
installed board packages before wiring the two devices together.
