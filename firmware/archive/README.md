# Firmware archive

This directory preserves historical code snapshots for traceability. It is not
the primary location for maintained or learner-facing firmware.

The 2024 prototype ZIPs contain two different revisions of the same Arduino
sketch. The `2024-12-07/project_x_prototype/` directory preserves a later
snapshot in the folder structure required by the Arduino IDE.
Keep these revisions until their differences have been reviewed and any useful
history has been converted into normal Git commits or documented releases.
