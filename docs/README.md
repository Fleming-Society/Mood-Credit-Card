# Documentation

- [`learner/`](learner/) contains public instructions, handouts and slides.
- [`instructor/`](instructor/) contains recovered agendas, session plans,
  preparation guidance, safeguarding notes and risk information.
- [`images/`](images/) contains project screenshots used by the root README.

Keep editable source files alongside exported PDFs whenever possible. Use a
year subfolder only for genuinely historical deliveries; maintain current
instructions at a stable path.
