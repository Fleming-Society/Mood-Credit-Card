# 2024 hardware archive

These directories preserve meaningful Project X PCB development stages from
the original source drop. Each revision is self-contained where the recovered
files permit it.

| Directory | Description |
| --- | --- |
| `early-mood-creditcard-v1.0/` | Early schematic and almost-empty PCB placeholder |
| `session-1-v1.1/` | First substantial Session 1 KiCad design and Gerbers |
| `session-1-v2/` | Later Session 1 design, Gerbers and legacy DipTrace source |
| `project-x-v3-portrait/` | Portrait 65 mm by 38 mm design revision |
| `project-x-v3.1/` | Later Project X design with Gerbers and DipTrace source |
| `learner-old/` | Earlier learner project, manufacturing output and laptop-specific board variant |
| `session-1-incomplete/` | Isolated project file retained because its board and schematic were absent |

Automatic backups, caches, lock files, user preferences and autosaves were
not copied into the public archive. Repeated board STEP exports and component
3D models were also excluded because their embedded or source-model licensing
has not been confirmed. The untouched source ZIP remains under ignored
`private/source-archive/`.

These revisions have not been revalidated. Do not manufacture from them or
present them as current teaching material.
