# Enclosure CAD

The recovered 2024 SolidWorks and STL files are grouped by development stage:

- [`2024/v3/`](2024/v3/) contains the v3 assembly, enclosure and back variants.
- [`2024/earlier-revisions/`](2024/earlier-revisions/) contains earlier
  enclosure and assembly files.

The SolidWorks assemblies may refer to component models that are not publicly
distributed because their licences are unconfirmed. Open the part files first
if an assembly reports missing references.

No preferred enclosure revision has been identified. Check dimensions,
connector access, print tolerances, board fit and sensor access before using
an STL for fabrication.
