# Manufacturing outputs

This directory is for generated, reviewed fabrication outputs. Editable source
belongs in `../kicad/`.

The `gerbers/` files were inherited from the 2024 project and are not currently
approved for a new production run. Once validated, publish a dated/versioned
bundle through a GitHub Release and record:

- source commit;
- KiCad version;
- board revision;
- ERC and DRC result;
- fabricator-specific settings, if any; and
- reviewer and approval date.
